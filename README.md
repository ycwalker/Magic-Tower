## 魔塔RPG

#### [Demo 地址](http://ycwalker.com/Magic-Tower/)

#### 涵盖经典魔塔中的所有游戏元素

## 技术实现

* 内部实现了一个微型的MVVM，实现了简单的插值绑定
* 使用`Typescript`面向对象方式编写
* `LocalStore`作为存档方式
* 精灵图和`CSS`动画的实现
